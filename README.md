## Komentorivi

Ohjelman voi suorittaa komennolla:

mvn compile exec:java -Dexec.mainClass=sudoku.Main

#### Testaus

Testit voi suorittaa komennolla:

mvn test

Testikattavuuden voi suorittaa komennolla:

mvn jacoco:report
